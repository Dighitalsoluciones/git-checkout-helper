# Git Checkout Helper

## Descripción

Extensión para generar comandos `git checkout` a partir de la salida de `git status`.

## Uso

1. Ejecuta el comando `Git Checkout Helper` desde la paleta de comandos.
2. Ingresa el nombre de la rama y pega la salida de `git status`.
3. Copia el comando generado y ejecútalo en tu terminal.

## Características

- Soporte para rutas con espacios
- Detección automática de archivos modificados/nuevos
- Copiado al portapapeles
